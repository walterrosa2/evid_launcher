# EVID IA Launcher

Central de Inteligência para Análise de Processos e Auditoria. Este painel centraliza o acesso a diversos módulos da plataforma EVID IA.

## Tecnologias
- Python
- Streamlit

## Como executar localmente
1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
2. Execute o app:
   ```bash
   streamlit run launcher_app.py
   ```

## Deploy no Railway
O projeto está configurado para deploy automático via GitHub utilizando o `Procfile` e `requirements.txt`.
